== Description ==

Live comment preview without page reboot. Works on jQuery.

**Attention:** *knowledge of CSS and HTML may require for using the plugin.*

**Features:**

* Live comment preview, which does not take extra space on a page.
* The simple HTML-editor (instruction on using the editor are on the plugin options page).
* You can customize the comment preview block as you wish. You can display: comment author name, avatar (static or dinamic using gravatar.com), comment date and time, comment text. All of this can be designed with CSS.
* You may don't connect the plugin's CSS file (optional), if you wish to reduce a number of queries to your server.
* The plugin's JS and CSS files are connected only on a pages with a comment form.

== Installation ==

1. Upload `jquery-comment-preview` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. If it's necessary, you can customize options in admin interface at the `Options -> jQuery Comment Preview` page.
4. If it's necessary, you can edit CSS styles for the comment preview block in the `jquery-comment-preview.css` file.
5. That's all.
